Beliefs & Learnings

  • Bug fixes > parameter tuning. Biggest wins: role misassignment fix (+4.53), agent_id normalization (+9.64), critic field mismatch.
  • Hub penalty helps independent agents. Keeps agents near hub where they can observe each other's captures. Removing it = 2x regression.
  • Local scores ≠ tournament scores. 18.38 local → 2.93 tournament. Cooperative team scoring is a different game.
  • Seed variance is extreme. 0.00–12.29 across seeds. Need 5+ seeds to evaluate.
  • Agents capture then lose. 8 friendly junctions at step 1000 → 0 by step 3000. Defense is the bottleneck.
  • PCO works when critic is correct. Bad signals (resource always 100) → bad patches. Fixed critic → meaningful proposals.
  • v14 score was matchmaking artifact. v22 (identical code) scored 2.41 vs v14's 6.23. Rating feedback loop inflated it.
  • No shared state. Agents are independent. Per-agent junction memory only.
